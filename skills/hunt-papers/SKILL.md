# Hunt Papers

Use the `arxiv-search` tool to find exact matching papers, foundational works, or recent advancements directly related to the contents of `TECHNIQUES.md`.

## Instructions
1. Read the `TECHNIQUES.md` file generated by the `scan-codebase` skill.
2. For each major architectural element or training pattern, form an optimal, targeted search query and call the `arxiv-search` tool.
3. Compile a deduplicated list of the most relevant papers found across all queries.
4. Output your findings precisely as a markdown file `PAPERS.md` placed directly in the root of the project.

## Output Format (PAPERS.md)
Format the output exactly as follows:
- **[arxiv ID]**: [Paper Title]
  - Related to technique: [Technique Name from codebase]
  - Relevance reason: [Brief explanation of why this paper applies]
