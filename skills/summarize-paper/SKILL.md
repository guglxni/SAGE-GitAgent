# Summarize Paper

Takes the `PAPERS.md` document and produces crisp, engineering-focused summaries for the highest-value papers detected.

## Instructions
1. Read the `PAPERS.md` file generated by the `hunt-papers` skill.
2. For each important arxiv ID found, execute the `fetch-abstract` tool to gather official metadata.
3. Parse and process the theoretical abstract into an highly actionable, engineering-focused summary tailored for developers.
4. Output your summaries precisely as a markdown file `SUMMARIES.md` placed in the root of the project.

## Output Format (SUMMARIES.md)
Format each summary precisely as follows:
### [Paper Title] ([arxiv ID])
- **What it does**: [One sentence explanation]
- **How to implement it**: [Implementation sketch or pseudo-code]
- **What it replaces**: [What older technique it replaces or improves upon]
